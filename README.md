# dotfiles

Minimalistic look using [Hyprland](https://hyprland.org/) for the tiling window manager, Waybar for the top bar, kitty & and zsh for the terminal and shell.

## Theme 

- Catppuccin Macchiato (System theme)
- Dracula (Icon theme)
